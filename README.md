依赖库：Flask 3.0.0, FlaskSQLAlchemy 3.1.1
先运行一次 app.py 创建数据库文件（会自动生成 grades.db）
你会看到类似输出：
  Serving Flask app 'app'
  Debug mode: on
  Running on http://127.0.0.1:5000 (Press CTRL+C to quit)
这说明程序已经成功启动。
在浏览器中访问生成网址你会看到成绩管理系统的仪表盘页面。
你可以进行的操作：
点击导航栏的 学生管理 / 课程管理 / 成绩管理 进入对应模块
添加学生：需要填写姓名、学号（不能重复）、班级
添加课程：需要填写课程名称、课程编号（不能重复）、学分
添加成绩：选择已有学生和课程，输入 0-100 的分数
成绩筛选：可以按学生或课程筛选，页面会实时显示平均分、最高分、最低分
编辑或删除：每个条目旁都有按钮，删除前会有确认提示

所有数据都会自动保存在 grade_system/grades.db 这个 SQLite 数据库文件中，程序关闭后再打开数据依然存在。
